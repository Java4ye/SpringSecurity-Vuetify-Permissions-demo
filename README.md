# 概要

该项目是一个很简单的前后端分离项目， 后台使用了SpringSecurity，前端使用了Vuetify，做了一个简单的动态路由，权限控制。😝

## 后端

主要是为了学习前后端分离的情况，SpringSecurity的使用 ，jdk11！😄

## 前端

Vuetify的学习，还有动态路由，权限的简单例子

## 启动

**本地**跑起来，在vue文件夹中，执行

1.  `npm insall` 
2.  `npm run serve` 

SpringSecurity也直接跑起来就行了。

如果后端端口不是8080，需要修改下vue.config.js文件，将端口改为对应的就行😄
